# Sammunat Landing Page

A production-quality landing page for a developer tool startup found at [Sammunat](https://sammunat-demo.vercel.app). 
Built as part of an emotional design and frontend engineering assignment.

## Project Overview

Sammunat ("Prosperity") is a fictional SaaS platform designed to help engineering teams ship code faster without burnout. 
The landing page demonstrates a clean, modern aesthetic with a focus on typography, spacing, and usability.

### Key Features
- **Modern UI/UX**: Clean layout, neutral color palette, and subtle interactions.
- **Sticky Navigation**: Smooth scrolling and easy access to sections.
- **Responsive Design**: Fully functional on mobile and desktop.
- **Custom Components**: Built from scratch without UI libraries.
- **Contact Form**: Frontend validation and success state handling.

## Tech Stack

- **Framework**: React.js 18 + Vite
- **Styling**: Vanilla CSS Modules + CSS Variables
- **Icons**: Inline SVG (No external libraries)
- **Deployment**: Vercel / Netlify Ready

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd sammunat-landing-page
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## Folder Structure

```
src/
├── components/    # Reusable UI components
├── styles/        # CSS Modules and global variables
├── App.jsx        # Main application layout
├── main.jsx       # Entry point
└── index.css      # Global resets and variables
```

## Deployment

This project is optimized for deployment on Vercel or Netlify.
simply connect the repository and deploy with default settings (Output directory: `dist`).
